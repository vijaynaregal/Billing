GST-Inclusive Invoice Generator

Create invoices with GST treated as inclusive: 5% when unit price is below ₹2500 and 18% when unit price is ₹2500 or more. Optional discount can be toggled and applied as a custom percentage before GST breakdown.

Launch Full Page
  • Add products with name, unit price (GST-inclusive), and quantity.
  • Choose payment method (Cash or Online).
  • Enable discount and set a custom percentage if needed.
  • Download the PDF invoice. If the download doesn’t start, check your browser’s popup/download settings.
If the embedded view doesn’t load, use the Launch Full Page button above.

Notes

  • Ensure invoice_generator.html is in the same directory as this index.html.
  • Some browsers block file downloads or popups; allow downloads for this page if prompted.
  • For best results, serve these files over a local server (e.g., using VS Code Live Server or python -m http.server).
